F.A.Q.

Q: Is this a remake of Super Mario Maker?
A: Sort of. This project started out as a simple SMB1 engine, but as soon as Super Mario Maker came out, I decided I just had to replicate the "stylized" SMB1 style in it (with shadows), then I ended up reamking various aspects from Super Mario Maker, including Costume Mushrooms (which are fully customizable).

Q: Are you going to add other game styles (SMB3, SMW, NSMBU)?
A: No. This fangame is only gonna have the SMB1 style, which is already the most content-packed style of Super Mario Maker anyways (with the costumes, weird Mario, etc.). If you want these styles just play the original game!

Q: What about a custom style or a style that hasn't been done before (i.e. SMB2)?
A: Probably not. But if there ever will be something like that, the difference will be probably just graphically. The folder of the game is already huge in size because of all music.

Q: Can "X" be a costume? / Can you add a costume of the character "X"?
A: While you are free to request any costume you'd like to see, the answer will be most likely no. Nothing stops you from creating your own costume, though, as you can customize all of them.

Q: Can you add this feature?
A: Like the costumes, I cannot guarantee every request will be granted, but feel free to request.

Q: The costumes are not working!!!
A: Remember that they have to be in a NUMBERED folder in the costumes folder within the content folder (located at %localappdata%\Super_Mario_ReMaker, there's a shortcut to it in the ZIP file).

Q: I can't place a flagpole!
A: For "safety", I decided to make it so you can only place a flagpole on top of normal blocks (letter B shortcut).

Q: The costumes I downloaded from the costume packs have a white background in-game!
A: Make sure you download the costume packs as ZIP from dropbox, and not every file individually.

Full editor
Featuring 9 Course Themes (Ground, Underground, Underwater, Castle, Ghost House, Ice, Desert, Athletic and Night) + one customizable theme.
Most elements
From SMB1 + some Super Mario Maker exclusives and a few elements from The Lost Levels, SMB Deluxe, and Mario Forever.
Powerups
All of the SMB1 style powerups (Mushroom, Fire Flower, Costume Mushroom, Weird Mario).
Special mushroom
Costume Mushroom with 100% customizable costumes.
Simple user interaction
Hotkeys for items and "shake" function (right click).
Special modes
In example "You vs Boo" from SMBDX.
